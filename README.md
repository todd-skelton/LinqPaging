# LinqPaging
Paging made easy with Linq syntax and extensions. Translates directly to SQL for database-side paging when using Entity Framework.

## Installation
### Package Manager
`Install-Package LinqPaging`

### .NET CLI
`dotnet add package LinqPaging`
